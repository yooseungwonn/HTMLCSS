공간 분할 태그 실습

block 방식

이것은 div 태그로 만든 공간입니다

div 태그는 블록 수준 요소로, 여러 요소를 그룹화하고 스타일을 적용할 때 사용됩니다.

줄 바꿈이 일어납니다.

블록 방식의 태그

  • div
  • h1 ~ h6
  • p
  • 목록 관련 태그
  • 테이블 관련 태그
  • form 관련 태그

여기는 div 태그 외의 내용입니다.

inline 방식

이것은 span 태그로 감싼 텍스트입니다.

span 태그는 인라인 요소로, 텍스트 또는 인라인 요소를 그룹화하고 스타일을 적용할 때 사용됩니다.

줄 바꿈이 일어나지 않습니다. 문맥의 흐름이 유지됩니다.

인라인 방식의 태그

  • span
  • a
  • input 태그
  • 글자 형식 태그들
